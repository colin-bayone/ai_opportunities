### Case Study Reference — Multimodal SFT for Field-Support Visual Assistant

**Origin:** Colin's prior org (Coherent/II-VI). This is NOT BayOne work. This is a case study from real work Colin did before joining BayOne.

**Scenario:** Global field service org supporting 8K+ complex industrial laser systems across 40+ countries. Downtime costs around $12K/hr. On-site expert dispatches are expensive, slow, and involve safety risks. Strong demand for faster on-site fault diagnosis.

**What was built:**
- Supervised fine-tuning (SFT) of a vision-language model on aligned image + text data (fault photos, schematics, repair steps)
- Azure AI data pipeline for ingestion, labeling, augmentation, model training, secure model registry
- Multilingual output (English, German, Mandarin) via Azure Translator
- PII and sensitive design redaction, safety tags, audit trail for regulatory compliance
- Post-deploy MLOps workflow with drift detection and periodic SFT refreshes every two weeks

**Solution details:**
- 15-week program, 12-person squad aided by client team
- 75k image-text pairs + 25k augmented variants
- LoRA adapters on text head, frozen vision backbone
- Tablet-based workflow where tech snaps photo, model returns top-3 ranked faults + safety-checked repair checklist, tech confirms/fixes
- Confirmed repairs feed back into training set with auto-curation bot scoring label quality nightly

**Results:**
- 50-60% reduction in ticket resolution time
- First-pass fix rate improved from 71% to 91% (+28%)
- 50% reduction in expert call-outs and travel costs
- 15% reduction in safety-procedure deviations
- +6 CSAT points from field techs
- SLA penalties cut by ~$3.2M annually

**Technical stack:** Azure AI Vision, Azure Machine Learning, Azure Blob Storage and Data Lake, Azure DevOps + GitHub Actions, Power BI dashboards. 93% top-1 fault-classification accuracy. Sub-2s inference latency. $0.04 per-inference average with cost recovery break-even at 3 months.

### Why This Case Study Is Relevant to Walmart

The core pattern maps well. Troy's team manages store technology infrastructure (POS, self-checkout, payment, IoT, associate devices) with a heavy operational and support workload. They want predictive outage detection and real-time monitoring. The Coherent case study demonstrates exactly this type of work at scale... field techs using AI to diagnose and resolve hardware/system issues faster, with measurable impact on resolution time, first-pass fix rates, and cost reduction.
