# Colin Moore | Director of AI

Colin Moore is an AI/ML engineering leader with deep experience building and deploying enterprise-scale AI systems that operate reliably under commercial-grade requirements. He currently serves as Director of AI at BayOne Solutions, where he leads technical strategy and manages AI/ML engineering teams.

Prior to BayOne, Colin was Global AI/ML Manager at Coherent Corporation, where he established and led the company's AI Center of Excellence. He managed a global team of 30+ AI engineers and data scientists distributed across the US, EU, and APAC, building a portfolio of 60+ AI initiatives that delivered over $30M in measurable annual impact.

Representative projects:

**Context-Aware Code Review System:** Designed and deployed an AI code review tool that addresses the core problem of noise over signal in automated review. Rather than reviewing files in isolation, the system ingests organizational coding standards, architectural patterns, and examples of both desirable and problematic code to build context. The tool integrates directly into developer workflows via VS Code and Visual Studio plugins, dramatically reducing false positives and increasing engineer trust in automated review feedback.

**Enterprise GenAI Platform:** Architected and deployed a FedRAMP-authorized Azure-hosted LLM platform serving 40,000+ users across Aerospace & Defense and global manufacturing operations. The system provides role-based access control through Entra/Azure AD, supports 25+ languages for global deployment, and runs on cost-optimized Kubernetes infrastructure.

**Multi-Model ML Systems:** Led development of production ML systems combining gradient-boosted models with reinforcement learning for continuous optimization, with SHAP-based explainability layers that allow engineers to understand why recommendations are made rather than treating models as black boxes.

**Data Pipeline Architecture:** Designed enterprise ETL and MLOps infrastructure processing approximately 10TB daily, assimilating 70+ on-premises engineering databases across 9 countries alongside Oracle Cloud infrastructure. The system handles reconciliation, synchronization, and data auditability across heterogeneous sources while enabling real-time streaming dashboards for operational visibility.

**Global Manufacturing Cost Optimization:** Led development of an AI system for a medical laser manufacturer operating four global assembly facilities, replacing annual spreadsheet-based planning with continuous optimization. XGBoost models predicted yields, tariffs, and landed costs; a mixed-integer linear programming optimizer generated nightly build allocations across sites while respecting export-control constraints. SHAP explainability allowed operations leaders to understand each recommendation. Results: landed-cost variance reduced from ±6% to ±1.8%, COGS decreased 5-6%.

**High-Stakes Process Optimization:** Built a multi-model ML system for advanced materials manufacturing where each production run cost $3M and took 3-4 months—with no ability to test changes on dedicated R&D equipment. Gradient-boosted models predicted outcomes from 150 sensor streams, temporal convolutional networks provided early-warning detection, and PPO reinforcement learning trained entirely within COMSOL/ANSYS digital twins validated recommendations before any real-world implementation. Domain experts retained final decision authority. Results: failure rate cut 50%, yield increased from 65% to 82%, $11M annual loss reduction.

Colin's early work was in biomedical engineering and embedded systems, building photoacoustic flowmetry systems using Total Internal Reflection Photoacoustic Spectroscopy (TIRPAS) and pulsed lasers for melanoma detection. He then transitioned into chemical engineering R&D, developing novel lithium-sulfur battery technology for electric vehicles. His work on immobilized chalcogen cathode materials—solving the polysulfide shuttling problem that limits lithium-sulfur battery cycle life—resulted in three patents and a technology sale to General Motors.

Colin has served as primary technical advisor to legal, compliance, and cybersecurity teams on AI deployment risks.

In this engagement, Colin will provide hands-on technical leadership on AI architecture and implementation.
