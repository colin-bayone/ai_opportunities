# Machine Learning Engineer

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring a Machine Learning Engineer for a high-visibility engagement with a Fortune 500 company. You will join an elite, fast-paced team building systems that drive real business impact—generative AI for product discovery, personalized recommendation engines, customer segmentation, and predictive modeling.

This is a hands-on role spanning the full model lifecycle: development, training, validation, deployment, and production monitoring. You will work with Azure's machine learning stack, build and operationalize feature stores, and implement both batch and real-time inference systems. We're looking for someone proactive who takes ownership and engages deeply with the team rather than waiting for direction.

You will partner with product managers, data scientists, enterprise architects, and business stakeholders to design pipelines and productionize models. This role demands strong engineering fundamentals and MLOps expertise.

## Key Responsibilities

### Model Development & Productionization (50%)

- Own the full model lifecycle: develop, train, validate, deploy, and monitor models in production
- Build and operationalize feature stores
- Implement batch and real-time inference solutions using PyTorch, TensorFlow, or Keras
- Deploy and optimize open-source LLMs using LLMOps practices
- Optimize model serving through frameworks like ONNX, TF Serving, or MLEAP
- Write efficient, well-organized code in an iterative, continual-release environment

### Infrastructure & Operations (30%)

- Architect and maintain production infrastructure on Azure
- Build CI/CD pipelines for model training, validation, and deployment
- Implement monitoring, logging, and A/B testing frameworks
- Deploy and operate containerized systems using Docker and orchestration tools
- Design APIs for model serving and integration with downstream systems
- Ensure production systems meet performance, reliability, and scalability requirements

### Team Collaboration (20%)

- Partner with product, data science, architecture, and business teams on new capabilities
- Participate in code reviews and uphold engineering best practices
- Document systems, pipelines, and operational procedures
- Communicate progress, blockers, and technical decisions clearly
