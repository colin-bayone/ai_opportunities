ML Platform Engineer

BayOne Solutions | Remote (Offshore)

Position Overview

BayOne Solutions is hiring an ML Platform Engineer for a high-visibility engagement with a Fortune 500 company. You will join an elite, fast-paced team as a full-stack machine learning professional responsible for both model development and the data infrastructure that powers it. This includes work on generative AI for product discovery, personalized recommendation engines, customer segmentation, and predictive modeling.

This is a senior hands-on role requiring end-to-end ownership across the entire machine learning stack. You will develop and deploy models, build and maintain the data pipelines that feed them, architect feature stores, and ensure reliable production operations. You will work across Azure's machine learning and data platforms, implementing both batch and real-time systems at scale.

You will partner with product managers, data scientists, enterprise architects, and business stakeholders to drive initiatives from concept through production. This role demands deep expertise in both machine learning engineering and data engineering, with the ability to move fluidly between them.

Key Responsibilities

Model Development & Productionization (40%)

  • Own the full model lifecycle: develop, train, validate, deploy, and monitor models in production
  • Build and operationalize feature stores that serve both training and inference
  • Implement batch and real-time inference solutions using PyTorch, TensorFlow, or Keras
  • Deploy and optimize open-source LLMs using LLMOps practices
  • Optimize model serving through frameworks like ONNX, TF Serving, or MLEAP
  • Design and implement A/B testing frameworks for model evaluation

Data Infrastructure & Pipelines (35%)

  • Design, build, and maintain data pipelines and ETL processes at scale
  • Work with Spark, Kafka, and other distributed data technologies
  • Build and optimize data models for analytical and machine learning workloads
  • Implement workflow orchestration for complex data and model pipelines
  • Ensure data quality, reliability, and timely delivery across the platform

Platform Operations & Infrastructure (25%)

  • Architect and maintain production infrastructure on Azure, spanning both ML and data systems
  • Build CI/CD pipelines for model training, data transformations, and deployment
  • Implement comprehensive monitoring, logging, and alerting across the platform
  • Deploy and operate containerized systems using Docker and Kubernetes
  • Design APIs and services for model serving and data access
  • Ensure production systems meet performance, reliability, and scalability requirements
