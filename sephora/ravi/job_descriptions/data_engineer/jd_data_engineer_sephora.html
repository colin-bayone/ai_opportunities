Data Engineer

BayOne Solutions | Remote (Offshore)

Position Overview

BayOne Solutions is hiring a Data Engineer for a high-visibility engagement with a Fortune 500 company. You will join an elite, fast-paced team building the data infrastructure that powers machine learning systems driving real business impact—generative AI for product discovery, personalized recommendation engines, customer segmentation, and predictive modeling.

This is a hands-on role focused on building and operating data pipelines, ETL processes, and the infrastructure that feeds production models. You will work with Azure's data stack, design scalable data architectures, and ensure reliable data flows from source systems through to feature stores and model training pipelines. We're looking for someone proactive who takes ownership and engages deeply with the team rather than waiting for direction.

You will partner with machine learning engineers, data scientists, enterprise architects, and business stakeholders to build the data foundation that enables model development and deployment. This role demands strong data engineering fundamentals with experience supporting MLOps workflows.

Key Responsibilities

Data Infrastructure & Pipelines (50%)

  • Design, build, and maintain data pipelines and ETL processes at scale
  • Work with distributed data technologies such as Spark, Kafka, Hadoop, or Databricks
  • Build and optimize data models for analytical and machine learning workloads
  • Ensure data quality, reliability, and timely delivery to downstream systems
  • Develop and maintain feature stores that serve production models
  • Write efficient, well-organized code in an iterative, continual-release environment

Platform & Operations (30%)

  • Architect and maintain data infrastructure on Azure
  • Build CI/CD pipelines for data workflows and transformations
  • Implement monitoring, logging, and alerting for data pipelines
  • Deploy and operate containerized systems using Docker and orchestration tools
  • Design APIs and services for data access and integration
  • Ensure production systems meet performance, reliability, and scalability requirements

Team Collaboration (20%)

  • Partner with machine learning engineers and data scientists to understand data requirements
  • Collaborate with architecture and business teams on data strategy
  • Participate in code reviews and uphold engineering best practices
  • Document systems, pipelines, and operational procedures
  • Communicate progress, blockers, and technical decisions clearly
