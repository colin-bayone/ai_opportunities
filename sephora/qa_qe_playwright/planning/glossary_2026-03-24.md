## Sephora-Specific Terms

| Term | Definition |
|------|-----------|
| **AMP** | Agentic Micro Pod. Sephora's new team structure replacing traditional pods. Lean 2-3 person teams focused on small, frequent enhancements. Demands fast-paced testing to match development velocity. |
| **Nova** | Sephora's homegrown AI platform, built by Nikhil's MCP and DevOps team. LiteLLM-based. Intended as the unified AI platform that all functions (including QE) will migrate onto. Name stands for something like "Next, Virtual Assistant." |
| **QE COE** | Quality Engineering Center of Excellence. Led by Deepika, ~5 people. Owns the QE agent roadmap. |
| **Advent** | AI platform Sephora adopted after DataIQ became too expensive. Claude was adopted as the model, Advent as the platform. Being superseded by Nova. |
| **DataIQ** | Earlier AI platform/tool used by Sephora QE. Abandoned due to cost ("became a lot more expensive, sort of an enterprise model"). Possibly Dataiku. |
| **MCP** | Model Context Protocol. Sephora requires MCP server integration for all new tools to enable seamless workflow integration. |
| **Regular pod** | Sephora's traditional team structure being replaced by AMP. Larger teams, less frequent release cycles. |

## Industry / Technical Terms

| Term | Definition |
|------|-----------|
| **Playwright** | Microsoft's browser automation framework. Sephora is migrating from Selenium to Playwright. POCs complete, new development already on Playwright. |
| **LiteLLM** | Open-source library for unified LLM API access. Foundation of Sephora's Nova platform. |
| **Relevancy testing** | Testing to measure the quality of agentic workflow outcomes. Verifying agents are producing correct results. A key concern for Vaibhav. |
| **ADA compliance testing** | Accessibility testing against Americans with Disabilities Act standards. Identified as a gap in Sephora's current QE coverage. |
| **State graph** | BayOne's approach to mapping a codebase deterministically: dependency mapping, module relationships, dead code detection. Plugs into CI/CD for targeted test triggering. |
