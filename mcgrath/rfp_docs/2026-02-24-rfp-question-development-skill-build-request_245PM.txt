в—Ҹ Gemini API keys are free and easy to get:

  1. Go to Google AI Studio: https://aistudio.google.com/
  2. Sign in with your Google account
  3. Click "Get API Key" in the left sidebar (or top menu)
  4. Click "Create API key" - you can create one for a new project or
  existing Google Cloud project
  5. Copy the key and set it:
  export GEMINI_API_KEY="your-key-here"

  Free tier includes:
  - 15 requests/minute for Gemini 2.0 Flash
  - 1,500 requests/day
  - No credit card required

  For the document ingestion use case, the free tier should be plenty unless
   you're processing hundreds of pages in rapid succession.
