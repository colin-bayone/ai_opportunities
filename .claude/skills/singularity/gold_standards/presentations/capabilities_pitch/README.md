# Gold Standard: Capabilities Pitch

**Source engagement:** Ariat foundational slides
**Created:** 2026-03-03
**Purpose:** Gold standard for BayOne AI capabilities pitch presentations. Three slides demonstrating how to present BayOne's service offerings to a prospect. Not a complete deck, but a complete capabilities pitch showing three distinct ways to frame what BayOne does.

## Files

| File | What It Demonstrates |
|------|---------------------|
| `slide_01_ai_strategy_innovation.html` | AI Strategy and Innovation: 3+2 staggered card grid covering developer productivity, enterprise automation, data & analytics, document intelligence, and agentic AI. Each card has an icon header, tagline, and 3 bullet items. |
| `slide_02_enterprise_ai_solutions.html` | Enterprise AI Solutions: 2x2 card grid with HR, Finance, Legal, Marketing verticals. Each card uses a 2-column item layout inside the card for density. |
| `slide_03_quality_engineering.html` | Quality Engineering: Chevron flow bar showing a 6-step process pipeline across the top, with a 3x2 detail card grid below mapping to each step. Unique layout pattern not found in other gold standards. |

## Color Palette

These slides use the **BayOne purple** palette (`--purple-darkest`, `--purple-accent`, `--purple-glow`), unlike the Masterminds deck (blue) and team status update deck (blue).

## Layout Patterns Demonstrated

- **Staggered card grid** (3 top + 2 bottom) — slide 01
- **2x2 card grid with 2-column items inside cards** — slide 02
- **Chevron flow bar + detail card grid** — slide 03 (also extracted to `layout_examples/chevron_flow_detail.html`)

## How to Use

When pitching BayOne's capabilities to a new prospect, these slides show how to organize service offerings into digestible categories with specific, concrete capabilities listed under each. They demonstrate different card grid arrangements for different amounts of content.
