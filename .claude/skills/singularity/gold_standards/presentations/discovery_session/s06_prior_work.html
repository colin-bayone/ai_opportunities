Prior Work

Understanding what was tried, what it was tested against, and what would have constituted success.

The Application Context

The approaches tried and the results they produced are much more meaningful when we understand the specific environment they were tested in.

What specific application or system was the prior effort running against?
What was that application supposed to do for its users?
How does improving detection and redaction help it function better?

What Was Tried

ML-based approaches were explored, rule-based models were considered, and a labeling exercise was evaluated.

What was the scope and intent of each attempt?
What data did each attempt run on?
What specifically fell short in each approach?

Defining Success

Not technical metrics, but what would have changed for the team if the prior effort had worked.

What was the prior effort supposed to deliver as an outcome?
What would have made the team say "this is working, let us keep going"?
What would have changed operationally for the engineers using the system?
