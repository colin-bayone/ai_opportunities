# Gold Standard: Discovery Session Presentation

**Source engagement:** Lam Research IP Protection
**Created:** 2026-04-06
**Purpose:** Gold standard for discovery session presentations. An 11-slide deck used to walk a client through discovery findings, present the proposed technical approach, and align on next steps. Demonstrates how to structure a discovery readout as a presentation.

## Structure

| Slide | File | Topic |
|-------|------|-------|
| 01 | `s01_title.html` | Title slide |
| 02 | `s02_core_challenge.html` | The core challenge being addressed |
| 03 | `s03_business_impact.html` | Business impact and urgency |
| 04 | `s04_use_cases.html` | Use cases under discussion |
| 05 | `s05_where_we_are.html` | Current state assessment |
| 06 | `s06_prior_work.html` | What's been tried before |
| 07 | `s07_poc_target.html` | Selecting the POC target |
| 08 | `s08_application_detail.html` | Understanding the selected application |
| 09 | `s09_hybrid_architecture.html` | Proposed hybrid detection architecture |
| 10 | `s10_azure_data_plane.html` | Common data plane and Azure integration |
| 11 | `s11_next_steps.html` | Next steps and action items |

## How to Use

When preparing a discovery readout presentation, use this deck as the structural reference. It demonstrates the narrative arc: problem framing (slides 2-4) → current state and prior work (5-6) → proposed approach (7-10) → next steps (11). This flow works for any engagement where you need to present discovery findings and a recommended path forward.
