# Gold Standard: RFP Proposal Presentation

**Source engagement:** McGrath RentCorp MSP RFP
**Created:** 2026-03-23
**Purpose:** Gold standard for RFP response presentations. A 29-slide proposal deck covering executive summary, solution architecture, transformation journey, AI strategy, case studies, risk mitigation, commercials, and closing. Demonstrates how to structure a comprehensive managed services proposal as a slide deck.

## Structure

The deck follows a standard proposal narrative flow:

| Slides | Section |
|--------|---------|
| 01 | Title |
| 02 | Executive summary |
| 04, 08 | Section dividers (Our Understanding, Proposed Solution) |
| 05-07 | Architecture, solution towers, scope summary |
| 09-13 | Proposed solution: transformation journey, AI strategy, enterprise AI, quality engineering, solution summary |
| 14-18 | Operations: snapshot, phase 1 discovery, operational maturity, quality audit, KPI metrics |
| 20 | Why BayOne |
| 21-24 | Case studies (HCM impl, HCM managed, ERP reporting, EBS support) |
| 25 | Risks and mitigation |
| 27-30 | Key asks, commercials, prerequisites, assumptions |
| 35-36 | Summary and closing |

## Color Palette

Purple BayOne palette (`--purple-darkest` through `--purple-glow`).

## How to Use

When responding to an RFP or building a managed services proposal, read through this deck to see how technical scope, commercial terms, case studies, and risk mitigation are structured as a cohesive presentation narrative. Pay attention to the section flow, the use of divider slides to break sections, and how case studies are integrated as credibility evidence.
