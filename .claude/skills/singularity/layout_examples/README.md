# Layout Examples

Visual vocabulary of unique slide layout patterns extracted from across all gold standard presentations. Each file demonstrates one layout pattern with real production content. Use this index to find the right pattern for your content, then open the file to see how it's built.

## Layout Index

### `title.html` — Dark Full-Bleed Title
**Source:** Masterminds deck
**Use when:** Opening slide of any presentation.
**What it looks like:** Full dark gradient background, centered title text, subtitle, presenter name, date. No header bar. BayOne logo rendered in gradient text.

### `agenda.html` — Horizontal Section Card Row
**Source:** Masterminds deck
**Use when:** Table of contents, roadmap overview, multi-track summary.
**What it looks like:** Row of section cards laid out horizontally, each acting as a chapter summary with an icon, section title, and description. Card top borders use gradient progression left to right.

### `profile.html` — Split Panel Profile
**Source:** Masterminds deck
**Use when:** Presenter bio, team member introduction, stakeholder profile.
**What it looks like:** Left panel (35%) has dark gradient with photo placeholder, name, title, and company. Right panel (65%) has a vertical stack of highlight cards, each with an icon, heading, and description.

### `split_concept.html` — Split Panel Concept
**Source:** Masterminds deck
**Use when:** Deep dive on a topic, explanation with supporting detail, any content that has a "framing" side and a "detail" side.
**What it looks like:** Left panel provides narrative context or a conceptual frame. Right panel has structured cards, lists, or details. Similar structure to profile but with content instead of a person.

### `three_column.html` — Three-Column Card Grid
**Source:** Masterminds deck
**Use when:** Comparing 3 items side-by-side, category breakdowns, overview of 3 related topics.
**What it looks like:** Three equal-width cards across the full slide width. Each card has an icon header, title, tagline, and bullet items. Gradient progression across the three cards.

### `grid_takeaway.html` — Card Grid with Takeaway Bar
**Source:** Masterminds deck
**Use when:** Enumerations of 4-6 items with a concluding insight, category breakdowns that need a summary.
**What it looks like:** Grid of cards (3+2 staggered or other arrangement) filling the content area, plus a dark takeaway bar at the bottom with a bold concluding statement. Lead text under the title sets up the grid.

### `closing.html` — Dark Full-Bleed Closing
**Source:** Masterminds deck
**Use when:** Final slide, Q&A, thank you.
**What it looks like:** Full dark gradient background, centered closing text, optional subtitle. Mirror of the title slide pattern with closing-specific content.

### `chevron_flow_detail.html` — Chevron Process Bar + Detail Cards
**Source:** Ariat capabilities pitch
**Use when:** Sequential process with details per step, pipeline visualization, methodology breakdown.
**What it looks like:** Horizontal chevron-shaped flow bar across the top showing 6 process steps with gradient progression. Below it, a 3x2 grid of detail cards where each card maps to a step in the flow bar. Each detail card has an icon, title, tagline, and 4 bullet items. Purple palette.
