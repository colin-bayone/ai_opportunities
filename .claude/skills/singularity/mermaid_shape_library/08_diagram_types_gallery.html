ZenUML (Alternative Sequence Diagram)

Keyword: zenuml — Code-like sequence diagram syntax (requires external plugin)

Plugin required: ZenUML is an alternative sequence diagram syntax that uses a code-like format. It requires the ZenUML plugin to be registered with mermaid and does not render with the standard CDN alone. See mermaid ZenUML docs for setup.
