Syntax Reference

Copy-paste syntax for each shape. The A is the node ID (use any name). The brackets determine the shape.

ShapeSyntaxBest For
RectangleA["Text"]General process steps, actions
RoundedA("Text")Events, AI/ML components, soft processes
StadiumA(["Text"])Start/end terminals, entry/exit points
SubroutineA[["Text"]]Predefined processes, build tools, function calls
CylinderA[("Text")]Databases, data stores, storage
CircleA(("Text"))Connectors, junction points
DiamondA{"Text"}Decisions, verification, branching
HexagonA{{"Text"}}Orchestrators, engines, central coordinators
ParallelogramA[/"Text"/]Input/output
Parallelogram ReverseA[\"Text"\]Alternative input/output
TrapezoidA[/"Text"\]Manual operations
Trapezoid ReverseA[\"Text"/]Alternative manual ops
Double CircleA((("Text")))Important connectors, emphasis
AsymmetricA>"Text"]Flags, signals, events
