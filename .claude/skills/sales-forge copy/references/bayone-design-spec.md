# BayOne Design Spec Reference

This reference links to the canonical BayOne design specification.

**Location:** `specs/bayone-design-spec.md` (project root)

## Quick Reference

### Brand Colors
- `--purple-darkest: #2e1065` - Deep purple for covers
- `--purple-dark: #4c1d95` - Gradient midpoint
- `--purple-mid: #5b21b6` - Table headers
- `--purple-bright: #6d28d9` - Gradient endpoint
- `--purple-accent: #a855f7` - Section numbers
- `--purple-glow: #e879f9` - Logo accent

### Key CSS Classes
| Class | Purpose |
|-------|---------|
| `.cover` | Full-page gradient cover |
| `.section-number` | Purple "01" labels |
| `.highlight-box` | Purple-bordered callout |
| `.stat-grid/.stat-card` | Metrics display |
| `.card` | Bordered content card |

### Typography
- Font: Inter (Google Fonts)
- H2: 32px, 700 weight
- Body: 15px, line-height 1.7
- Section numbers: 12px uppercase, letter-spacing 2px

### Style Rules
**DO:**
- Numbered sections (01, 02, 03)
- Ample whitespace
- Natural paragraphs

**DON'T:**
- Emojis
- Excessive bullets
- Multiple color schemes

For full specification, read `specs/bayone-design-spec.md`.
