### Steady State Flowchart Detail

**L0 / L1 Flow**
- Start
- User support email / VM
- Create ~~Freshservice~~ ticket
- Is this supported ticket
  - If No: Inform user -> Close Ticket
  - If Yes: (moves to L2/L3 Flow)

**L2 / L3 Flow**
- (From L0/L1 'Yes') -> Acknowledge user
- Can you resolve?
  - If Yes: Assign ticket to yourself & resolve -> Close Ticket
  - If No: Can other agent resolve?
    - If Yes: Assign to an agent who can resolve
    - If No: Assign to L3 team -> Resolve & Close Ticket -> Update Knowledge DB
- (Parallel flow from L0/L1 'Yes') -> Update Knowledge DB
