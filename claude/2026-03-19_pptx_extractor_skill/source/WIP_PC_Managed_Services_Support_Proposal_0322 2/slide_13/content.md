*(The content below is contained within a flowchart diagram)*

### L0 / L1 Flow
- Start
- User support email / VM
- Create Fresh Service ticket
- Is this supported ticket
    - If **No**:
        - Inform user
        - Close Ticket
    - If **Yes**: (Flow continues to L2 / L3)

### L2 / L3 Flow
- Update Knowledge DB
- Acknowledge user
- Can you resolve ?
    - If **Yes**:
        - Assign ticket to yourself & resolve
        - Close Ticket
    - If **No**:
        - Assign to an agent who can resolve
- Can other agent resolve?
    - If **Yes**:
        - Resolve & Close Ticket
        - (updates Knowledge DB)
    - If **No**:
        - Assign to L3 team
        - Resolve & Close Ticket
        - Update Knowledge DB
- 3rd party ticket
    - Co-ordinate with 3rd party
    - Resolve & Close Ticket
    - (updates Knowledge DB)
