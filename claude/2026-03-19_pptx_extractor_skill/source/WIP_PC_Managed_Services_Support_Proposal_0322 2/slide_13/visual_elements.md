# Slide 13 - Visual Elements

This slide contains a single, complex diagram and supporting text elements.

- **Type:** Flowchart Diagram
- **Position:** Covers the majority of the slide, from top to bottom, offset to the right of a vertical banner.
- **Description:** The diagram illustrates a "High Level Support Process Flow." It is divided into two horizontal swimlanes within a dashed border. The top swimlane is for the "L0 / L1 Flow" and the bottom for the "L2 / L3 Flow". The flowchart uses color-coding to distinguish between two teams: grey shapes for "McGrathTeam" (in the L0/L1 flow) and purple shapes for "BayOne Team" (in the L2/L3 flow). Standard flowchart shapes are used: rounded rectangles for start/end points, a parallelogram for user input, rectangles for processes, diamonds for decisions, and a cylinder for a database.
