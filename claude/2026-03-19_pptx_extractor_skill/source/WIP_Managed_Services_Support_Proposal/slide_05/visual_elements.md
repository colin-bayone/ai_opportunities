# Slide 5 - Visual Elements

This slide contains a variety of visual elements, primarily icons and branding graphics.

1.  **Icon: AI Driven IT Support Operations**
    *   **Type:** Icon
    *   **Position:** Top-left, next to the service title.
    *   **Description:** A purple line-art icon depicting a cloud with a gear inside it. Below the cloud, a horizontal platform supports two vertical arrows, one pointing up and one pointing down, indicating data transfer or operations.

2.  **Icon: QA & Testing Automation**
    *   **Type:** Icon
    *   **Position:** Top-center, next to the service title.
    *   **Description:** A purple line-art icon featuring a large circular speech bubble containing a smaller rectangular speech bubble (with lines suggesting text) and an exclamation mark. To the right is a smaller, separate speech bubble containing a question mark.

3.  **Icon: UI / UX Services**
    *   **Type:** Icon
    *   **Position:** Top-right, next to the service title.
    *   **Description:** A purple line-art icon showing a stylized browser or application window. Inside the window are code symbols: `< / >`. Two arrows point diagonally outward from the top-left and bottom-right corners, suggesting scalability or responsiveness.

4.  **Icon: Application Modernization**
    *   **Type:** Icon
    *   **Position:** Bottom-left, next to the service title.
    *   **Description:** A purple line-art icon of a computer monitor with a gear symbol in the center. The monitor is connected by lines to four nodes, resembling a network or circuit board. On the far left and right are stylized Wi-Fi signal symbols.

5.  **Icon: DevOps / SRE Services**
    *   **Type:** Icon
    *   **Position:** Bottom-center, next to the service title.
    *   **Description:** A purple line-art icon showing two circular arrows forming a continuous, horizontal infinity-like loop. Inside each of the two loops is a gear symbol, representing a continuous process or cycle.

6.  **Icon: PMO / EPMO Services**
    *   **Type:** Icon
    *   **Position:** Bottom-right, next to the service title.
    *   **Description:** A purple line-art icon depicting a cyclical process. Four smaller icons (a document, a clock, and a clipboard with a checkmark) are arranged in a circle, connected by curved arrows to illustrate a workflow or lifecycle.

7.  **Header Bar**
    *   **Type:** Graphic
    *   **Position:** Top of the slide, below the main title.
    *   **Description:** A solid, thick horizontal line in a magenta color.

8.  **Logo: BAYONE**
    *   **Type:** Logo
    *   **Position:** Bottom-left, in the footer area.
    *   **Description:** The word "BAYONE" in a bold, sans-serif, dark gray font. The "BAY" is in a regular bold weight, while "ONE" is in a heavier, black weight.

9.  **Footer Background**
    *   **Type:** Graphic
    *   **Position:** Bottom of the slide.
    *   **Description:** A graphic element with a light purple background and flowing, wave-like shapes in shades of purple and pink.
