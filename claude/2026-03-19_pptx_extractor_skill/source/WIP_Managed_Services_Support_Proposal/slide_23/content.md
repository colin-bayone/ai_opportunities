### Steady State Phase Flowchart Detail

**L0 / L1 Flow**
*   Start
*   User support email / VM
*   Create Freshservice ticket
*   Is this supported ticket?
    *   No -> Inform user -> Close Ticket
    *   Yes -> (Moves to L2 / L3 Flow)

**L2 / L3 Flow**
*   (From "Is this supported ticket?" -> Yes)
*   Acknowledge user
*   Can you resolve?
    *   Yes -> Assign ticket to yourself & resolve -> Close Ticket
    *   No -> Can other agent resolve?
        *   Yes -> Assign to an agent who can resolve -> (Loops back to "Can you resolve?")
        *   No -> Assign to L3 team -> Resolve & Close Ticket -> Update Knowledge DB -> (Loops back to start of L2/L3 flow, before "Acknowledge user")
*   Update Knowledge DB (Also feeds into the L2/L3 flow before "Acknowledge user")
