# High Level Support Process Flow

**BayOne End-to-end ownership**

---

**L0 / L1 Flow**

*   Start
*   User support email / VM
*   Create Fresh Service ticket
*   Is this supported ticket
    *   No -> Inform user -> Close Ticket
    *   Yes -> (Moves to L2 / L3 Flow)

**L2 / L3 Flow**

*   Acknowledge user
*   Can you resolve?
    *   Yes -> Assign ticket to yourself & resolve -> Close Ticket
    *   No -> Assign to an agent who can resolve
*   Can other agent resolve?
    *   Yes -> Resolve & Close Ticket -> Update Knowledge DB
    *   No -> Assign to L3 team -> Resolve & Close Ticket -> Update Knowledge DB
    *   (If 3rd party ticket) -> Co-ordinate with 3rd party -> (Connects back to 'Resolve & Close Ticket' in L3 path)

*   Update Knowledge DB (This is also a starting point for some flows)
