# Slide 12 - Visual Elements

This slide has a clean, corporate design with a distinct color scheme and graphical elements.

- **Type:** Header Bar
- **Position:** Top of the slide, below the main title.
- **Description:** A solid, thin horizontal line in a bright magenta color that spans the width of the slide.

- **Type:** Columnar Layout
- **Position:** Main body of the slide.
- **Description:** The content is organized into three distinct vertical columns. Each column has a purple header bar and a light lavender content area.

- **Type:** Header Boxes
- **Position:** At the top of each of the three columns.
- **Description:** Three rectangular boxes with a solid medium-purple background. Each contains white text identifying the section and its frequency.
    - Left box text: "Strategic | Quarterly"
    - Middle box text: "Tactical | Monthly"
    - Right box text: "Operational | Daily / Weekly"

- **Type:** Text Color Styling
- **Position:** Throughout the content boxes.
- **Description:** Specific words are colored for emphasis or branding. "McGrath:" is in purple text, and "BayOne:" is in magenta text. All other text is black.

- **Type:** Horizontal Separators
- **Position:** Inside each of the three content columns.
- **Description:** Thin, dark purple horizontal lines that divide the "who" (roles) from the "what" (topics) within each column.

- **Type:** Footer Graphic
- **Position:** Bottom of the slide.
- **Description:** A footer element with a graphical background and text. The background consists of layered, wavy shapes in shades of light purple and pink.

- **Type:** Logo
- **Position:** Bottom-left of the slide, within the footer.
- **Description:** The "BAYONE" logo. The text "BAY" is in a dark purple/black sans-serif font. The "O" is replaced by a purple-to-magenta gradient circle. "NE" is in the same font as "BAY".
- **Text:** BAYONE

- **Type:** Slogan/Hashtag
- **Position:** Bottom-left, to the right of the logo.
- **Description:** The text "#TheFutureWorksHere." in a magenta sans-serif font.

- **Type:** Page Number
- **Position:** Bottom-right corner.
- **Description:** The number "12" in a grey, sans-serif font.
