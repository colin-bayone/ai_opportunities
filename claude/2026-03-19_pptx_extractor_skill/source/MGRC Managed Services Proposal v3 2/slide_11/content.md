# BayOne Escalation Matrix

| Escalation Level | Escalate to (Role) | Possible Triggers |
| :--- | :--- | :--- |
| **Level 1** | Project Manager<br>Program Manager | SLA breach risk, recurring defects, minor timeline delays, resource constraints |
| **Level 2** | Delivery Manager<br>cc: Account Manager | SLA breach, high priority issues not resolved within SLAs, quality issues |
| **Level 3** | Account Manager, Delivery Mgr<br>cc: US Sales Head, CTO | Customer dissatisfaction, formal complaints, contractual risks |
| **Level 4** | CEO, CTO<br>cc: Account Mgr, Delivery Mgr | Strategic account risk, legal/compliance issues, significant financial impact |

A structured, multi-level escalation framework ensures rapid issue resolution, clear accountability and proactive risk mitigation.
