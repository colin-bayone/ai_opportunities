# Service Quality Audit Process

---

**On-boarding Phase**
* Application on-boarding
* New feature / enhancements
* AI transformation / automations / efficiencies

**Steady State Phase**
1. Conduct process review
2. Identify process gaps / inconsistency
3. Implement improved / standardized process
4. Perform root cause analysis
5. Update knowledge repository

* 3rd party quality control audit process in place at each phase
* Identify & close process gaps & data inconsistency
* Continuous improvement cycle for better service quality
