# High Level Support Process Flow
---
## BayOne End-to-End Ownership

### L0 / L1 Flow
1. User support email / VM received
2. Create FreshService ticket
3. Route: Is this a supported ticket?
4. If Yes → L2/L3 Flow
5. If No → Inform user & close ticket

### L2 / L3 Flow
1. Acknowledge user & assign ticket
2. Can you resolve? → Resolve & close
3. If No → Assign to L3 team
4. 3rd party ticket → Coordinate vendor
5. Update Knowledge DB & close

---
*Key: Automation & self-help opportunities will be identified and implemented to improve cycle time & productivity*
