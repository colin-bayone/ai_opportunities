# Slide 5 - Visual Elements

This slide contains a title, a 2x3 grid of service offerings, and a footer with a logo and decorative graphics.

1.  **Icon: AI Driven IT Support Operations**
    *   **Type:** Icon
    *   **Position:** Top-left content block, to the left of the title.
    *   **Description:** A purple line-art icon depicting a cloud with a gear inside it. Two vertical arrows, one pointing up and one pointing down, pass through the cloud. This symbolizes managed, cloud-based IT processes.

2.  **Icon: QA & Testing Automation**
    *   **Type:** Icon
    *   **Position:** Top-center content block, to the left of the title.
    *   **Description:** A purple line-art icon featuring a large speech bubble containing horizontal lines (representing text) and a circle with an exclamation point. Overlapping this is a smaller speech bubble with a question mark. This represents communication, queries, and issue identification in testing.

3.  **Icon: UI / UX Services**
    *   **Type:** Icon
    *   **Position:** Top-right content block, to the left of the title.
    *   **Description:** A purple line-art icon showing a browser window with code brackets (`</>`) inside. Arrows point from the window's corners to smaller square outlines, suggesting responsive design across different screen sizes.

4.  **Icon: Application Modernization**
    *   **Type:** Icon
    *   **Position:** Bottom-left content block, to the left of the title.
    *   **Description:** A purple line-art icon of a computer monitor with a gear in the center. It is surrounded by a circuit-board pattern with wireless signal symbols on the left and right, symbolizing modern, connected applications.

5.  **Icon: DevOps / SRE Services**
    *   **Type:** Icon
    *   **Position:** Bottom-center content block, to the left of the title.
    *   **Description:** A purple line-art icon with two interlocking gears inside a circular arrow loop, representing the continuous, cyclical nature of DevOps and SRE practices.

6.  **Icon: PMO / EPMO Services**
    *   **Type:** Icon
    *   **Position:** Bottom-right content block, to the left of the title.
    *   **Description:** A purple line-art icon showing a document, a clock, and a checkbox connected by circular arrows. This visualizes the process of planning, time management, and task completion in project management.

7.  **Logo: BAYONE**
    *   **Type:** Logo
    *   **Position:** Bottom-left corner of the slide.
    *   **Description:** The word "BAYONE" in a bold, sans-serif, dark grey typeface.

8.  **Graphic: Footer Wave**
    *   **Type:** Decorative Graphic
    *   **Position:** Across the bottom of the slide.
    *   **Description:** A subtle, abstract wave-like shape composed of overlapping layers in shades of light purple and magenta.

9.  **Graphic: Title Underline**
    *   **Type:** Decorative Graphic
    *   **Position:** Directly under the main title "BayOne Service Offerings".
    *   **Description:** A solid, thick horizontal line in a magenta/purple color.
