Automation Engineer

BayOne Solutions | On-site (San Jose, CA)

Position Overview

BayOne Solutions is hiring an Automation Engineer for a high-visibility engagement with a Fortune 500 technology company. You will build and integrate automation systems across a large-scale CI/CD pipeline, working with tools like Apache Airflow, Jenkins, and Bazel to improve developer productivity and pipeline reliability.

This is a hands-on engineering role. You will write production code daily, integrate with complex enterprise systems, and work directly with client engineering teams. You will collaborate closely with a Technical Lead who sets overall architecture, a Senior AI Solutions Engineer who builds AI-powered features, and an offshore team that supports implementation.

The client environment is on-prem, Python-heavy, and involves integration with mature enterprise systems including multiple validation gates, orchestration tools, and proprietary internal platforms. You should be comfortable navigating complex CI/CD infrastructure, working with incomplete documentation, and building integrations that operate reliably at scale.

Key Responsibilities

CI/CD Integration and Automation (60%)

  • Build integrations with Apache Airflow, Jenkins, and other CI/CD orchestration tools
  • Develop data pipelines that capture telemetry from CI/CD systems, including test results, build status, and pipeline metrics
  • Create automation that connects disparate systems and normalizes data across different schemas
  • Implement intelligent automation features that reduce manual intervention in pipeline operations
  • Work with Bazel and other build systems to understand and integrate with the client's build infrastructure
  • Integrate with internal tools including commit approval systems, developer platforms, and monitoring dashboards
  • Ensure integrations perform reliably in an on-prem environment with appropriate error handling and retry logic
  • Write production-quality Python code with comprehensive testing and documentation

Backend Development (20%)

  • Develop backend services that support dashboards, automation features, and AI capabilities
  • Create APIs that expose pipeline data to front-end applications and AI systems
  • Build data models that accurately represent CI/CD state across multiple source systems
  • Implement caching, queuing, and other patterns appropriate for high-volume pipeline data
  • Ensure services meet performance and reliability requirements in a production environment

Client Interface (15%)

  • Serve as a day-to-day technical interface with client engineering teams alongside the Senior AI Solutions Engineer
  • Work directly with client CI/CD and DevOps teams to understand their systems and integration points
  • Participate in technical discussions about pipeline architecture and automation opportunities
  • Build productive working relationships with client engineers who own the systems you integrate with
  • Navigate a large enterprise organization to obtain access, documentation, and technical context
  • Represent BayOne professionally in all client interactions

Technical Collaboration (5%)

  • Work under the Technical Lead's direction on architecture decisions and technical approach
  • Collaborate with the Senior AI Solutions Engineer on integration points between automation and AI features
  • Coordinate with offshore engineers, providing context on CI/CD systems and reviewing their work
  • Surface technical risks, blockers, and dependencies early to avoid surprises

Required Qualifications

CI/CD and Automation Experience

  • 3+ years of hands-on experience with CI/CD systems and pipeline automation
  • Strong experience with at least one major orchestration tool (Apache Airflow strongly preferred, or equivalent like Prefect, Dagster, or Luigi)
  • Experience with Jenkins, including pipeline configuration, plugin development, or API integration
  • Understanding of build systems and build automation (Bazel, Make, Gradle, or similar)
  • Practical knowledge of CI/CD patterns including gates, validation checks, and deployment pipelines
  • Experience building integrations between CI/CD tools and other systems
  • Familiarity with pipeline observability: monitoring build health, tracking metrics, and diagnosing failures

Software Engineering

  • 4+ years of professional software engineering experience
  • Strong Python proficiency, including building backend services, APIs, and data processing pipelines
  • Experience with data pipeline development: ETL processes, data transformation, and schema management
  • Solid database skills including SQL, schema design, and working with both relational and time-series data
  • Experience with API design and integration with third-party systems
  • Familiarity with containerization (Docker) and deployment in enterprise environments
  • Experience working in large, complex codebases with multiple contributors

Communication and Client Skills

  • Demonstrated ability to work directly with clients or stakeholders in a technical capacity
  • Strong written and verbal communication skills
  • Ability to work with client engineering teams to understand their systems and requirements
  • Comfortable discussing technical tradeoffs and explaining integration approaches
  • Professional demeanor suitable for representing a consulting firm to enterprise clients

Working Style

  • Self-directed and able to make progress with incomplete information
  • Detail-oriented; integration work requires precision and careful error handling
  • Comfortable debugging issues across multiple systems with limited documentation
  • Collaborative mindset; works well with distributed teams across time zones

AI-Assisted Development

  • Proficiency with AI pair programming tools (Claude Code, Cursor, GitHub Copilot, or similar) is expected
  • Ability to leverage AI assistance to accelerate development and maintain high output
  • Comfort integrating AI tools into daily workflow

Preferred Qualifications

  • Direct experience with Apache Airflow, including DAG development and Airflow API integration
  • Experience with Grafana for monitoring, dashboards, or data visualization
  • Background in developer productivity, internal tooling, or platform engineering
  • Experience in consulting, professional services, or client-facing engineering roles
  • Experience working in on-prem enterprise environments with security and access constraints
  • Experience building AI-powered products or working alongside AI engineers
  • Familiarity with log analysis, observability tools, or pipeline debugging
  • Experience with GitHub APIs, webhooks, or GitHub-based automation
