# Agentic AI Specialist

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring an Agentic AI Specialist to design and implement autonomous systems for a major enterprise client. You will build self-healing capabilities for CI/CD infrastructure, developing agents that can detect, diagnose, and remediate pipeline failures with minimal human intervention.

This role requires building systems that take real actions in production environments. You will be responsible for governance frameworks, safety mechanisms, and autonomous decision-making logic.

## Key Responsibilities

### Autonomous System Design (50%)

- Design and build autonomous decision-making systems that take real actions in production
- Implement governance frameworks defining what can be auto-corrected vs. what requires human approval
- Build safety mechanisms and rollback capabilities for automated actions
- Create action triggers that integrate with orchestration tools (Airflow, Jenkins)
- Design resilient systems that handle failure modes gracefully
- Define escalation paths for scenarios requiring human intervention

### Agent Development (35%)

- Develop production-grade autonomous agents using Python
- Implement tool-calling patterns using agent frameworks (LangChain, LangGraph, or similar)
- Integrate agents with CI/CD systems, monitoring tools, and internal APIs
- Build robust error handling and fallback mechanisms
- Deploy AI systems in on-prem environments

### Technical Leadership & Collaboration (15%)

- Provide technical direction on AI architecture decisions
- Review AI-related code and approaches
- Coordinate on system integrations and data pipelines
- Participate in governance discussions with the client
- Document agent behaviors, decision logic, and operational procedures

## Required Qualifications

- 5-8+ years of experience building AI/ML systems in production environments
- Deep experience building agentic AI systems that take autonomous actions in production
- Experience designing and implementing governance frameworks for AI systems
- Strong understanding of system design patterns for autonomous AI in enterprise environments
- Experience with agent frameworks (LangChain, LangGraph, or similar)
- Extremely strong Python developer, able to work at the system level
- Experience deploying AI systems on-prem
- Understanding of failure modes and how to build resilient autonomous systems
- Bachelor's degree in Computer Science, Engineering, or equivalent experience

## Preferred Qualifications

- Experience building self-healing or auto-remediation systems
- Background in reliability engineering, SRE, or production operations
- Prior work on CI/CD, DevOps, or infrastructure automation with AI
- Proficiency with AI-assisted development tools (Claude Code, Cursor, or similar)
- Experience with monitoring and observability systems
- Familiarity with Jenkins, Apache Airflow, or similar CI/CD tooling

Candidates with prior experience defining governance frameworks for autonomous AI systems will be strongly considered.
