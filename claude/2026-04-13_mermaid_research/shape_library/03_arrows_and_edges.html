Line Styles with Arrows

The three core line styles: solid (-->), dotted (-.->), and thick (==>). Solid = current/confirmed flow, dotted = proposed/optional, thick = primary/emphasized path.

graph LR A1["Source"] -->|"-->"| B1["Solid Arrow"] A2["Source"] -.->|"-.->"| B2["Dotted Arrow"] A3["Source"] ==>|"==>"| B3["Thick Arrow"]

Open Links (No Arrowhead)

Lines without arrowheads, useful for bidirectional associations or neutral connections.

graph LR C1["Node"] ---|"---"| D1["Solid Open"] C2["Node"] -.-|"-.-"| D2["Dotted Open"] C3["Node"] ===|"==="| D3["Thick Open"]

Arrowhead Variants

Beyond the standard pointed arrow, mermaid supports circle (--o) and cross (--x) terminators.

graph LR E1["Node"] -->|"--> Arrow"| F1["Target"] E2["Node"] --o|"--o Circle"| F2["Target"] E3["Node"] --x|"--x Cross"| F3["Target"]

Bidirectional Arrows

Arrows with terminators on both ends, for two-way communication or mutual relationships.

graph LR G1["Node A"] <-->|"<-->"| H1["Node B"] G2["Node A"] o--o|"o--o"| H2["Node B"] G3["Node A"] x--x|"x--x"| H3["Node B"]

Labeled Edges

Two syntax options for adding labels to edges: pipe syntax and inline syntax.

graph LR L1["Source"] -->|"Pipe Syntax"| L2["Target"] L3["Source"] -- "Inline Syntax" --> L4["Target"] L5["Source"] -.->|"Dotted + Label"| L6["Target"] L7["Source"] == "Thick + Label" ==> L8["Target"]

Invisible Links for Spacing

Use ~~~ to create invisible connections that influence layout without drawing a visible line. Useful for aligning nodes that have no logical connection.

graph LR I1["Node A"] ~~~ I2["Node B"] I1 --> I3["Connected to A"] I2 --> I4["Connected to B"]

Without the invisible link ~~~, Node A and Node B might stack vertically. The invisible link encourages them to sit side by side.
