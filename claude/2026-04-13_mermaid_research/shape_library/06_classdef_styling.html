ClassDef Syntax

Define styles with classDef, then apply using :::className inline or class A,B,C className for batch application.

classDef codeRepo fill:#eff6ff,stroke:#2563eb,stroke-width:2px,color:#1e3a5f classDef deprecated fill:#e2e8f0,stroke:#94a3b8,stroke-width:1px,color:#475569,stroke-dasharray:5 5 %% Inline application A["Node A"]:::codeRepo B["Node B"]:::deprecated %% Batch application class C,D,E codeRepo

Edge Styling with linkStyle

Use linkStyle N to style individual edges by their index (0-based, in the order they appear in the diagram source). Use linkStyle default to style all edges.

graph LR A["Start"] --> B["Step 1"] B --> C["Step 2"] C --> D["Step 3"] D --> E["End"] linkStyle 0 stroke:#2563eb,stroke-width:2px linkStyle 1 stroke:#16a34a,stroke-width:3px linkStyle 2 stroke:#dc2626,stroke-width:2px,stroke-dasharray:5 5 linkStyle 3 stroke:#7c3aed,stroke-width:4px

Edge 0 (blue, 2px) — Edge 1 (green, 3px) — Edge 2 (red, dashed) — Edge 3 (purple, 4px). Index is determined by the order edges appear in the mermaid source.

linkStyle default — Style All Edges at Once

Use linkStyle default to apply a uniform style to every edge in the diagram, instead of targeting each by index.

graph LR A["Start"] --> B["Step 1"] B --> C["Step 2"] C --> D["Step 3"] D --> E["End"] linkStyle default stroke:#7c3aed,stroke-width:2px

All four edges are styled uniformly with purple 2px stroke via a single linkStyle default declaration.

Batch Application with class Statement

Instead of applying :::className to each node individually, use class A,B,C className to apply a style to multiple nodes in one statement.

graph LR classDef highlight fill:#fcd34d,stroke:#b45309,stroke-width:2px,color:#78350f classDef muted fill:#e2e8f0,stroke:#94a3b8,stroke-width:1px,color:#475569 A["Node A"] B["Node B"] C["Node C"] D["Node D"] E["Node E"] A --> B --> C --> D --> E class A,B,C highlight class D,E muted

Nodes A, B, C are all styled amber via class A,B,C highlight. Nodes D, E are styled gray via class D,E muted. Useful when many nodes share a style.

Corner Radius with rx

The rx property controls how rounded the corners of rectangle nodes are. Higher values create more rounding. Note: the CSS override .node rect { rx: 8 } in the page stylesheet may override per-node rx values — this example shows the property in isolation.
