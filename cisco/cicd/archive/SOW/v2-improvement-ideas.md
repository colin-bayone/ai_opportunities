# SOW HTML v2 - Improvement Ideas

Working file: `SOW-Building-Nexus-9000-switches-v2.html`

---

## Proposed Improvements

### 1. Simplify Footer
**Current:** Footer shows "Confidential" + full document title + "Page X of 12"
**Proposed:** Reduce to just "Confidential" + "Page X of 12" (remove verbose filename)
**Rationale:** The title is already on every page header area; footer is cluttered

---

### 2. Add Indentation for Nested Content
**Current:** All content is flush left
**Proposed:** Indent subsection content (2.1, 2.2, etc.) slightly from their parent sections
**Rationale:** Creates visual hierarchy; easier to scan document structure

---

### 3. Improve Field Label Formatting (Page 2)
**Current:** "Currency: INR" as plain text paragraphs
**Proposed:** Use a subtle definition list style or light table for the metadata fields
**Rationale:** Better visual organization for the General Information / Timeline / etc. sections

---

### 4. Add Subtle Table Header Styling
**Current:** Table headers have no background
**Proposed:** Very light gray background (#f8f8f8) on table header rows
**Rationale:** Helps distinguish headers from data; still minimal/professional

---

### 5. Improve SLA Table Value Alignment
**Current:** Values in SLA tables are left-aligned
**Proposed:** Keep left-aligned but add slightly more padding for breathing room
**Rationale:** Tables feel cramped; small spacing improvement

---

### 6. Increase Line Height in Dense Text
**Current:** Line height is 1.4
**Proposed:** Increase to 1.5 or 1.6 for body paragraphs
**Rationale:** Improves readability without adding bulk

---

### 7. Add Visual Separator Before Major Sections
**Current:** Sections run together with just spacing
**Proposed:** Add a subtle top border or extra margin before sections 1, 2, 3, Financials
**Rationale:** Creates clearer document structure

---

### 8. Improve Signature Block Layout
**Current:** Signature lines use inline-block spans
**Proposed:** Use a cleaner table-based or flexbox layout with better alignment
**Rationale:** More professional appearance for the signature area

---

### 9. Reduce Cover Page Spacing
**Current:** Large gaps using margin-bottom: 2in on logo
**Proposed:** Tighten spacing slightly so cover page is more balanced
**Rationale:** Current layout feels sparse; better vertical rhythm

---

### 10. Style the Instruction Text Differently
**Current:** Instruction text like "[Include at least two SLAs...]" is just italic
**Proposed:** Add a light background or border to make it clearly stand out as a note/instruction
**Rationale:** Helps distinguish template instructions from actual content
