# Backend / Integration Engineer

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring a Backend / Integration Engineer to build data pipelines, integrations, and backend services for a major enterprise client. You will connect to existing enterprise systems, build robust data pipelines, and create backend infrastructure.

This is a core implementation role. You will be hands-on from Day 1, building the systems that power visibility dashboards, AI features, and automation capabilities.

## Key Responsibilities

### Data Pipelines & Integration (60%)

- Build telemetry systems to capture test runs, pass/fail status, duration, and coverage data
- Create data pipelines to store, process, and surface results
- Integrate with enterprise systems including Jenkins, Airflow, Grafana, and internal tools
- Normalize data across different systems and schemas
- Build the unified data layer for AI features and dashboards

### Backend Development (30%)

- Develop backend services using Python
- Create APIs for front-end applications and internal services
- Ensure services can run reliably in an on-prem environment
- Write clean, documented, and well-tested code

### Collaboration (10%)

- Daily syncs with the on-site technical lead
- Coordinate with front-end developers on API contracts
- Support AI engineers with data requirements for AI features
- Proactive communication about blockers and progress across time zones

## Required Qualifications

- 3-5+ years of professional software engineering experience
- Strong Python experience
- **Apache Airflow experience required**
- REST API integration experience
- Database experience (querying, schema design, data modeling)
- Experience building data pipelines
- Comfortable with CI/CD concepts and tooling
- Direct API integration experience
- Ability to work independently with guidance from the technical lead
- Proactive communicator, especially across time zones
- Bachelor's degree in Computer Science, Engineering, or equivalent experience

## Preferred Qualifications

- Prior DevOps experience or familiarity with CI/CD workflows and automation
- Experience with Jenkins, Grafana, or similar DevOps tooling
- Proficiency with AI-assisted development tools (Claude Code, Cursor, or similar)
- Familiarity with AI/ML concepts
- Experience working with enterprise systems
- Experience working in on-prem deployment environments
