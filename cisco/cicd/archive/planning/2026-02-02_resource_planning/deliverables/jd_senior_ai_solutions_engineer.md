# Senior AI Solutions Engineer

**BayOne Solutions | On-site (San Jose, CA)**

## Position Overview

BayOne Solutions is hiring a Senior AI Solutions Engineer for a high-visibility engagement with a Fortune 500 technology company. You will design and build AI-powered systems that improve developer productivity across a large-scale CI/CD pipeline, including intelligent chat interfaces, automated failure diagnosis, and AI-driven analysis tools.

This is a hands-on engineering role. You will write production code daily, work directly with client engineering teams, and help shape the technical direction of AI features. You will collaborate closely with a Technical Lead who sets overall architecture, an Automation Engineer who handles CI/CD integration, and an offshore AI team that supports implementation.

The client environment is on-prem, Python-heavy, and involves integration with mature enterprise systems including Apache Airflow, Jenkins, and proprietary internal tools. You should be comfortable navigating complex codebases, working with incomplete documentation, and building solutions that operate reliably at scale.

## Key Responsibilities

### AI Solution Development (60%)

- Design and build AI-powered features using large language models, including conversational interfaces that allow developers to query system status in natural language
- Develop intelligent failure diagnosis capabilities that analyze logs, identify root causes, and suggest fixes based on historical patterns and codebase understanding
- Implement AI-driven analysis tools that surface insights about pipeline health, test coverage, and developer workflows
- Build robust prompt engineering solutions and manage context windows effectively for production workloads
- Work with LangChain, LangGraph, or similar frameworks to create maintainable AI pipelines
- Write production-quality Python code with comprehensive testing, error handling, and observability
- Integrate AI capabilities with existing enterprise systems through direct API integration
- Ensure AI features perform reliably in an on-prem environment without cloud AI service dependencies

### Client Interface (25%)

- Serve as a day-to-day technical interface with client engineering teams alongside the Automation Engineer
- Participate in requirements gathering sessions, translating client needs into technical specifications
- Demo AI features to stakeholders, gather feedback, and iterate based on real-world usage
- Build productive working relationships with client engineers, earning trust through reliable delivery
- Navigate a large enterprise organization to obtain access, documentation, and technical context
- Communicate technical concepts clearly to both engineers and non-technical stakeholders
- Represent BayOne professionally in all client interactions

### Technical Collaboration (15%)

- Work under the Technical Lead's direction on architecture decisions and technical approach
- Collaborate with the Automation Engineer on integration points between AI features and CI/CD systems
- Coordinate with offshore AI engineers, providing context and reviewing their work
- Participate in code reviews, technical discussions, and architecture planning sessions
- Surface technical risks, blockers, and dependencies early to avoid surprises
- Document technical decisions and maintain clear records of AI system behavior

## Required Qualifications

### AI and LLM Experience

- 3+ years of hands-on experience building AI-powered features, with at least 1 year focused on LLM-based systems
- Demonstrated ability to take AI features from prototype to production
- Strong understanding of prompt engineering, including context management, few-shot learning, and output parsing
- Experience with AI orchestration frameworks such as LangChain, LangGraph, or equivalent
- Practical knowledge of LLM limitations, failure modes, and mitigation strategies
- Experience building conversational AI or chat-based interfaces
- Understanding of retrieval-augmented generation (RAG) patterns and vector databases

### Software Engineering

- 5+ years of professional software engineering experience
- Strong Python proficiency, including building backend services, APIs, and data processing pipelines
- Experience with API design and direct integration with third-party systems
- Solid understanding of software testing practices, including testing AI systems with non-deterministic outputs
- Familiarity with observability practices: logging, monitoring, and debugging production systems
- Experience working in large, complex codebases with multiple contributors

### Communication and Client Skills

- Demonstrated ability to work directly with clients or stakeholders in a technical capacity
- Strong written and verbal communication skills
- Ability to explain technical concepts to non-technical audiences
- Comfortable presenting work, gathering feedback, and iterating based on input
- Professional demeanor suitable for representing a consulting firm to enterprise clients

### Working Style

- Self-directed and able to make progress with incomplete information
- Comfortable with ambiguity and changing requirements
- Able to balance hands-on coding with client-facing responsibilities
- Collaborative mindset; works well with distributed teams across time zones

### AI-Assisted Development

- Proficiency with AI pair programming tools (Claude Code, Cursor, GitHub Copilot, or similar) is expected
- Ability to leverage AI assistance to accelerate development and maintain high output
- Comfort integrating AI tools into daily workflow

## Preferred Qualifications

- Experience with agentic AI patterns, autonomous systems, or AI systems that take actions
- Background in CI/CD tooling, developer productivity, or platform engineering
- Familiarity with Apache Airflow, Jenkins, or similar orchestration and CI tools
- Experience in consulting, professional services, or client-facing engineering roles
- Experience working in on-prem enterprise environments with security and access constraints
