# AI Engineer

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring an AI Engineer to build production AI features for a high-visibility engagement with a Fortune 500 technology company. You will develop conversational interfaces, AI-powered dashboards, and intelligent features that improve developer productivity across a large-scale CI/CD pipeline.

This is a hands-on engineering role. You will write production code daily, working closely with a Technical Lead and Senior AI Solutions Engineer who set technical direction. You will also collaborate with an Agentic AI Engineer on advanced AI capabilities and an Automation Engineer on integration points.

The client environment is on-prem, Python-heavy, and requires AI features that operate reliably without cloud AI service dependencies. You should be comfortable building production AI systems, working with LLM frameworks, and delivering features that real users depend on.

## Key Responsibilities

### AI Feature Development (65%)

- Build conversational interfaces that allow developers to query system status, pipeline health, and test results in natural language
- Develop AI-powered dashboard components that surface intelligent insights about developer workflows
- Implement chat-based features that integrate with multiple backend systems to provide unified answers
- Work with LLMs via LangChain, LangGraph, or similar frameworks to create maintainable AI pipelines
- Build robust prompt engineering solutions with appropriate context management and output parsing
- Implement retrieval-augmented generation (RAG) patterns to ground AI responses in system data
- Write production-quality Python code with comprehensive testing, error handling, and observability
- Ensure AI features perform reliably in an on-prem environment

### Dashboard and UI Development (20%)

- Build dashboard interfaces that visualize pipeline health, test coverage, and developer productivity metrics
- Develop front-end components that surface AI-generated insights in an intuitive way
- Create interactive features that allow users to drill down into AI analysis
- Work with FastAPI or Django to build backend services that support dashboard functionality
- Collaborate with the team on UI/UX decisions for AI-powered features

### Collaboration (15%)

- Work under the direction of the Senior AI Solutions Engineer and Technical Lead
- Coordinate with the Agentic AI Engineer on integration points between conversational AI and autonomous systems
- Support the Automation Engineer with AI requirements for pipeline integrations
- Participate in code reviews, technical discussions, and architecture planning
- Communicate progress, blockers, and technical decisions clearly across time zones
- Document AI system behavior and maintain clear technical records

## Required Qualifications

### AI and LLM Experience

- 2+ years of hands-on experience building AI-powered features, with meaningful exposure to LLM-based systems
- Experience taking AI features from development to production deployment
- Strong understanding of prompt engineering, including context management, few-shot learning, and structured output parsing
- Experience with AI orchestration frameworks such as LangChain, LangGraph, or equivalent
- Practical knowledge of LLM limitations and how to build reliable systems despite non-deterministic outputs
- Understanding of retrieval-augmented generation (RAG) patterns
- Familiarity with conversational AI patterns and chat interface development

### Software Engineering

- 3+ years of professional software engineering experience
- Strong Python proficiency, including building backend services and APIs
- Experience with web frameworks (FastAPI, Django, or Flask)
- Front-end development experience sufficient to build dashboards and interactive features (React, Vue, or similar)
- Understanding of API design and integration with third-party systems
- Solid testing practices, including strategies for testing AI systems with variable outputs
- Experience working in codebases with multiple contributors

### Communication and Working Style

- Strong written communication skills, particularly important for remote collaboration
- Proactive communicator who surfaces blockers and progress without being asked
- Comfortable working with distributed teams across time zones
- Self-directed and able to make progress with guidance from senior engineers
- Detail-oriented; AI features require careful attention to edge cases and failure modes

### AI-Assisted Development

- Proficiency with AI pair programming tools (Claude Code, Cursor, GitHub Copilot, or similar) is expected
- Ability to leverage AI assistance to accelerate development and maintain high output
- Comfort integrating AI tools into daily workflow

## Preferred Qualifications

- Experience building chat interfaces or conversational AI products
- Background in dashboard development or data visualization
- Familiarity with CI/CD concepts and developer productivity tooling
- Experience with vector databases (Pinecone, Weaviate, ChromaDB, or similar)
- Experience working in on-prem environments or enterprise deployments
- Background in NLP concepts: text processing, parsing, search, and classification
- Experience with monitoring and observability for AI systems
