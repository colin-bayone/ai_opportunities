AI Engineer

BayOne Solutions | Remote (Offshore)

Position Overview

BayOne Solutions is hiring an AI Engineer to build production AI features for a major enterprise client. You will develop a natural language chat interface, failure diagnosis systems that analyze logs and identify root causes, and AI-assisted debugging tools. This is an engineering role focused on building and deploying AI features that work reliably in production.

Key Responsibilities

AI Feature Development (60%)

  • Build a natural language interface that answers developer questions about PR status, pipeline health, and test results across multiple backend systems
  • Develop failure diagnosis capabilities that analyze CI/CD logs, identify root causes, and suggest remediation steps
  • Implement AI-assisted debugging that helps engineers triage gate failures
  • Build persona-based views that surface different insights for developers vs. release leads
  • Work with LLMs via LangChain/LangGraph and direct API integration
  • Deliver production-quality AI features with appropriate testing and documentation

Deployment & Infrastructure (25%)

  • Deploy AI services in an on-prem environment
  • Stand up and manage AI infrastructure independently
  • Optimize AI workloads for performance and reliability at scale
  • Monitor and maintain AI services in production

Collaboration (15%)

  • Work with backend engineers on data requirements for AI features
  • Coordinate with UI developers on surfacing AI outputs
  • Participate in team syncs and communicate progress
  • Proactively surface technical challenges and propose solutions

Required Qualifications

  • 4-6+ years of experience building production AI/NLP systems
  • Deep expertise with generative AI and LLMs
  • Strong NLP background including text processing, parsing, and search
  • LangChain and LangGraph experience required
  • Direct API integration with LLMs
  • Expertise in prompt engineering and LLM application architecture
  • Extremely strong Python developer
  • Experience deploying and operating AI services on-prem
  • Ability to deploy, configure, and maintain LLM-based services without dedicated infrastructure support
  • Bachelor's degree in Computer Science, Engineering, or equivalent experience

Preferred Qualifications

  • Prior work deploying LLMs in production environments
  • Background in building search, text processing, or NLP applications
  • Proficiency with AI-assisted development tools such as Claude Code
  • Experience with monitoring and observability for AI systems
  • Familiarity with CI/CD concepts and DevOps tooling
