# Agentic AI Engineer

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring an Agentic AI Engineer to design and build autonomous systems for a high-visibility engagement with a Fortune 500 technology company. You will develop self-healing capabilities for CI/CD infrastructure, creating agents that can detect, diagnose, and remediate pipeline failures with minimal human intervention.

This role focuses on AI systems that take real actions in production environments. You will design governance frameworks that define what actions are safe to automate, build safety mechanisms that prevent unintended consequences, and implement autonomous decision-making logic that operates reliably at enterprise scale.

You will work closely with a Technical Lead who sets overall architecture, a Senior AI Solutions Engineer who builds related AI features, and an AI Engineer who develops conversational interfaces. Together, the team delivers an integrated AI platform that improves developer productivity through intelligent automation.

The client environment is on-prem, Python-heavy, and involves integration with Apache Airflow, Jenkins, and other CI/CD systems. Autonomous actions must operate safely within enterprise security and compliance constraints.

## Key Responsibilities

### Autonomous System Design (45%)

- Design and build autonomous decision-making systems that take real actions in production CI/CD environments
- Develop governance frameworks that define what failures can be auto-remediated versus what requires human approval
- Create classification systems that categorize failures by type, severity, and remediation confidence
- Build safety mechanisms including action limits, circuit breakers, and automatic rollback capabilities
- Design escalation paths for scenarios where autonomous action is inappropriate or confidence is low
- Implement audit logging that provides complete visibility into what actions were taken and why
- Define boundaries between autonomous action and human oversight based on risk assessment
- Work with client stakeholders to establish trust in autonomous systems through transparency and incremental rollout

### Agent Development (35%)

- Develop production-grade autonomous agents using Python and agent frameworks (LangChain, LangGraph, or similar)
- Implement tool-calling patterns that allow agents to interact with CI/CD systems safely
- Build integrations with Apache Airflow, Jenkins, and other orchestration tools to enable automated retries, restarts, and corrections
- Create agents that can analyze failure patterns, determine root causes, and select appropriate remediation actions
- Implement robust error handling, timeout management, and fallback mechanisms
- Build monitoring and observability for agent behavior, including action success rates and intervention frequency
- Deploy and operate AI systems in on-prem environments without cloud service dependencies
- Optimize agent performance to minimize latency between failure detection and remediation

### Technical Collaboration (20%)

- Work under the direction of the Technical Lead on architecture decisions and system design
- Coordinate with the Senior AI Solutions Engineer on integration between autonomous systems and AI analysis features
- Collaborate with the AI Engineer on shared AI infrastructure and patterns
- Support the Automation Engineer with requirements for CI/CD integrations that enable autonomous actions
- Participate in code reviews, technical discussions, and architecture planning
- Document agent behaviors, decision logic, governance rules, and operational procedures
- Communicate progress, blockers, and technical decisions clearly across time zones
