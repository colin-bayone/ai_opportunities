## Key Findings

1. **The CICD opportunity is confirmed but smaller than expected.** $100K/quarter vs. $150-200K from Set 01. No explanation for the drop. Rahul frames it as renewable and growable.

2. **BayOne has a credibility problem.** Two consecutive consultant failures (Gaurav, then Sarang) in the same Cisco group. This is likely BayOne's last chance. Anand's ground rules (startup mindset, no staff aug) are partly a reaction to these failures.

3. **Sarang's failure was structural, not personal.** Colin's assessment: "I don't think there was any chance for the poor guy to be successful." No GitHub access, stuck in junior dev role, zero communication from US team. This reveals a Cisco culture where remote/offshore people are ignored by default.

4. **On-site presence is non-negotiable.** Both Rahul and Colin agree. Cisco culture requires walking up to people. The on-site person should be an interface layer (bridge to offshore team), not a pure developer who gets pigeonholed.

5. **Colin identifies two critical skill gaps.** (1) Agentic AI at the system level — not prompt engineering, but actual system-level automation. (2) Airflow experience — "the single biggest thing to connect to."

6. **Cisco's development practices are dysfunctional.** Code pasting in WebEx, not using GitHub properly, Srinivas's domain doesn't follow standard CI/CD workflows. Standard CI/CD experience won't transfer.

7. **Colin is new to BayOne costing.** Needs standard rates and margin guidance. This is his first resource plan at the company.

## Decisions Made

- Colin will write the full resource plan (A-F, one year, onshore + offshore) and JDs by Monday
- Amit will cost it
- Start without Sarang; bring him in consulting-side only if Cisco asks
- Colin's Jan 9-20 Bay Area trip is booked; will attempt in-person discovery
