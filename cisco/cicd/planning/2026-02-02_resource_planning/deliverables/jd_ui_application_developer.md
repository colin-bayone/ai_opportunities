# UI / Application Developer

**BayOne Solutions | Remote (Offshore)**

## Position Overview

BayOne Solutions is hiring a UI / Application Developer to own the user-facing parts of a major enterprise client project. You will build dashboards, data visualizations, and conversational interfaces. This role works across the stack, building the backend routes and integration work needed to wire everything together.

You must be comfortable working in Python backend code alongside your front-end work.

## Key Responsibilities

### Front-end Development (50%)

- Build dashboards and data visualizations for CI/CD pipeline metrics
- Develop conversational UI components for AI-powered features
- Create intuitive, functional user interfaces
- Deliver polished UI from design specs or requirements

### Backend Integration (40%)

- Write backend routes and API endpoints that serve the front-end
- Integrate with backend APIs for data retrieval and display
- Wire together front-end and backend components
- Work in Python for backend logic using FastAPI or Django
- Understand how data flows through an application

### Collaboration (10%)

- Coordinate with backend and AI engineers on API contracts and data requirements
- Participate in team syncs and communicate progress
- Take direction from the on-site technical lead

## Required Qualifications

- 2-4+ years of experience building web applications end-to-end
- Strong experience with modern front-end frameworks, React preferred
- Dashboard and data visualization experience
- Comfortable with Python and able to write API routes and backend logic
- Experience integrating front-end with backend services
- Some experience with ORMs and databases
- Proactive communicator across time zones
- Detail-oriented
- Bachelor's degree in Computer Science, Engineering, or equivalent experience

## Preferred Qualifications

- Experience with REST APIs and backend integration
- Django + Bootstrap 5 experience is a plus
- Proficiency with AI-assisted development tools (Claude Code, Cursor, or similar)
- Experience working with distributed teams
- Flexibility with frameworks and willingness to learn new technologies based on project requirements
