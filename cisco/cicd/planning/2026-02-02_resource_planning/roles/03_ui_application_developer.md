# Role: UI / Application Developer (Offshore)

**Location:** Offshore (India)

**Reports to:** Senior Engineer (Onshore), with oversight from Colin Moore (Director of AI)

**Relationship to Team:** Owns the user-facing parts of the system but is not a pure front-end developer. This person works across the stack - they build dashboards and interfaces, but they also do the backend integration work needed to wire everything together. Works collaboratively with the backend engineer and AI/ML engineer.

---

## Why This Role Exists

The engagement requires dashboards for visibility, data visualizations, and eventually a chat interface. Someone needs to own the UI layer, but that work doesn't exist in isolation - it requires connecting to backend services, integrating APIs, and sometimes building the backend routes that serve the front-end.

This is not a role for someone who only writes HTML, CSS, and basic JavaScript. This person needs to be comfortable working in Python backend code alongside their front-end work.

---

## Day-to-Day Responsibilities

**Front-end Development:**
- Build dashboards and data visualizations
- Develop the chat interface UI
- Create intuitive, functional user interfaces
- Work with design specs or requirements to deliver polished UI

**Backend Integration:**
- Write backend routes and API endpoints that serve the front-end
- Integrate with APIs built by the backend engineer
- Wire together front-end and backend components
- Comfortable working in Python (e.g., FastAPI, Django)

**Collaboration:**
- Work closely with backend engineer on API contracts and data requirements
- Coordinate with AI/ML engineer on chat interface integration
- Participate in team syncs and communicate progress
- Take direction from onshore senior engineer

---

## Required Skills

**Front-end:**
- Strong experience with modern front-end frameworks (React preferred, or Vue)
- Dashboard and data visualization experience

**Backend:**
- Comfortable with Python
- Can write API routes and backend logic
- Experience integrating front-end with backend services

**AI and Tooling:**
- Proficient with AI pair programming tools beyond basic Copilot (Claude Code, Cursor, or similar)
- Expected to use AI tools daily to accelerate development

**Working Style:**
- Can own the UI layer with guidance from the team lead
- Collaborative; works well with backend and AI engineers
- Proactive communicator across time zones
- Detail-oriented; UI work requires polish
