# Role: AI Engineer (Offshore)

**Location:** Offshore (India)

**Reports to:** Senior Engineer (Onshore), with oversight from Colin Moore (Director of AI)

**Relationship to Team:** Owns the AI-powered features of the system. Works closely with the backend engineer (who provides the data layer) and the UI developer (who builds the interfaces that surface AI capabilities). This is a senior technical role requiring deep AI expertise.

---

## Why This Role Exists

The engagement includes AI-powered features: a chat interface with natural language understanding, failure diagnosis that analyzes logs and suggests fixes, and AI-assisted debugging. Someone needs to build these capabilities - not as research or prototypes, but as production features that work reliably.

This is an engineering role, not a research or data science role. The focus is on building and deploying AI features that deliver value.

---

## Day-to-Day Responsibilities

**AI Feature Development:**
- Build the intelligence behind the chat interface (query understanding, response generation)
- Develop failure diagnosis capabilities (log analysis, root cause identification, suggested fixes)
- Implement AI-assisted debugging features
- Work with LLMs via LangChain/LangGraph and direct API integration

**Integration:**
- Work with backend engineer to access the data AI features need
- Coordinate with UI developer on how AI outputs are surfaced
- Ensure AI features integrate cleanly into the broader system

**Deployment:**
- Deploy AI services in an on-prem environment
- Should be able to stand up and deploy AI services independently
- Understand the infrastructure requirements for AI workloads

**Collaboration:**
- Participate in team syncs and communicate progress
- Take direction from onshore senior engineer
- Proactive about surfacing challenges with AI implementation

---

## Required Skills

**AI Expertise (Expert Level):**
- Deep experience with generative AI and LLMs
- Strong NLP background (text processing, parsing, search)
- LangChain and LangGraph experience required
- Direct API integration with LLMs (not just cloud-managed AI services)
- Understands prompt engineering, but goes well beyond it
- Can build production AI features, not just prototypes

**NOT Required:**
- Computer vision
- Traditional ML/statistical modeling

**Python:**
- Extremely strong Python developer
- This is non-negotiable; AI work is Python-heavy

**Deployment / Infrastructure:**
- Experience deploying AI services on-prem
- Able to stand up and deploy AI services independently without needing infrastructure hand-holding
- Understands how to run AI workloads at scale

**AI Tooling:**
- Proficient with AI pair programming tools (Claude Code, Cursor, or similar)
- Expected to use AI tools daily

---

## Experience Level

**Seniority:** Senior (4-6+ years), Expert in AI

- Extensive experience building production AI/NLP features
- Prior work with LLMs in real applications (not just experiments)
- Experience with on-prem deployment preferred
- Background in text processing, search, or NLP applications

---

## What Success Looks Like
