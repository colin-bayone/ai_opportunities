🔌 Modularity

  • Plug in Pulse directly if it's already a service app
  • Multiple connectors can feed the same DB in parallel

⚡ Pulse Integration Path

If Pulse is already a deployed service app with a registered access token → skip building the ingestion layer and wire MCP directly onto their existing DB. Pending confirmation of deployment status and token availability from Naga / Justin.
