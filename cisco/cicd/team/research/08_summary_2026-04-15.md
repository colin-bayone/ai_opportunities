## What is genuinely new since prior sets

### First explicit three-slide architecture framework

Set 07 introduced the Considerations-page concept and the decoupled service-layer architecture in general terms. Set 08 is where Colin articulated the full three-slide structure explicitly: current state framed as POC, gaps and considerations (including security, privacy, access control as dimensions beyond compute and scale), and recommended DeepSight-compatible unified-layer state. ### Saurav's blast-radius security framing

This is the first time blast-radius language appears in the team research chain. Saurav extended the WebEx scraping architecture with a space-level containment argument: scrape once per WebEx space and share across users, so that any attack or data leak is constrained to that space's data only. This adds a new framing dimension (security / blast radius) on top of the compute-efficiency argument for decoupled architecture.

### Whisper repositioned as fallback, not primary

Set 07 established the three-tier A/B testing framework for Whisper (Scribbler) versus WebEx native transcription. Set 08 repositions Whisper entirely: use WebEx native as primary, trigger Whisper only when WebEx transcription fails (non-English speakers, unclear audio). Colin based this on direct observation that WebEx transcripts sometimes return "the speaker was not talking clearly or was not talking in English and transcripts are not available." This cuts the compute cost argument for Whisper significantly and reframes the whole A/B narrative toward Srinivas.

Colin shared eight linked HTML files with Saurav via chat containing a comprehensive library of Mermaid 11.2 capabilities: shapes, text formatting, Font Awesome and Feather icons, reusable class definitions, subgraphs, and diagram types (class, Gantt, pie, mind map, milestone tracker, git graph). Known limitation: JavaScript padding issue on cut-off rendering (trivial fix).

This is an explicit shift of architecture ownership from Colin to Saurav for the WebEx track, consistent with Set 07's weekly-deliverable pattern.
