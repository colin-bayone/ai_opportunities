# Shareable Reports Bundle

This folder is a one-time handoff bundle that collects the current repository analysis artifacts into one place.

## Start Here

- `EPNM-EMS-CNC/consolidated_report.md`
  Combined portfolio report covering both EPNM and EMS-CNC.
- `EPNM-EMS-CNC/summary.json`
  Combined machine-readable summary for both families.
- `repo-inventory/repository_inventory.md`
  Repository inventory and organization plan.
- `repo-inventory/repository_inventory.json`
  Machine-readable repository inventory.

## Folder Layout

- `EPNM/`
  Family-level EPNM summary artifacts.
- `EPNM/tree-reports/`
  Per-repository tree reports for EPNM repos.
- `EMS-CNC/`
  Family-level EMS-CNC summary artifacts.
- `EMS-CNC/tree-reports/`
  Per-repository tree reports for EMS-CNC repos.
- `EPNM-EMS-CNC/`
  Combined cross-family report and summary.
- `repo-inventory/`
  Inventory of cloned/current repositories and target organization.
