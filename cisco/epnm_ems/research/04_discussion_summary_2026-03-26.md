## Key Takeaway

The $500K price point is outcome-based with clear margin floors (40% July, 30% December). The negotiation framework uses scope reduction as the only price lever. However, the pricing model was built on per-screen effort assumptions that may not account for the Set 03 scope reframe from full-stack conversion to classic view toggle. The POC will calibrate the real per-screen effort.

## Critical Flag

The pricing language references "full vertical slice (UI + backend)" while Set 03 established that the backend stays unchanged. This inconsistency needs resolution — either the pricing needs recalibration for the lighter scope, or the "full-stack" language is being used loosely to include the minor API adjustments Selva described.
