## What Changed

### 1. From Scope to Numbers
Set 03 defined what the work is (classic view toggle). Set 04 translated that into a financial model: $500K, outcome-based, three scenarios (July/December/discount), with margin targets and team structure.

### 2. Pricing Uses Pre-Reframe Language
Set 04 references "full vertical slice (UI + backend)" and "250+ screens converted, full-stack, end-to-end." This is the Set 01/02 framing, not the Set 03 reframe. The pricing was likely built on assumptions that predate the scope clarification, or the "full-stack" language is being used loosely. This is flagged as a potential inconsistency that should be resolved.

### 3. Team Structure Defined
Set 03 discussed the need to start looking for India resources. Set 04 formalized this: Colin at 30%, 1 US engineer, 3-8 offshore engineers depending on timeline.

### 4. Negotiation Framework Established
Set 03 acknowledged the budget question without discussing numbers. Set 04 built the complete negotiation playbook: no July discount, max 6% December discount, scope reduction as the primary lever.
